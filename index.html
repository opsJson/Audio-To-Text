Privacy Policy

A extensão Audio to Text for Whatsapp Web respeita e valoriza a privacidade dos seus usuários. Esta política de privacidade descreve como tratamos as informações que coletamos quando você utiliza nossa extensão.

Collection and Use of Information

A extensão Audio to Text for Whatsapp Web foi desenvolvida para converter áudios recebidos no Whatsapp em texto, facilitando a comunicação textual para os usuários. Ao utilizar nossa extensão, podemos coletar e processar o áudio que você escolher enviar para transcrição.

Utilizamos a API Whiser da OpenAI para transcrever o áudio em texto. Quando você envia um áudio, ele é transmitido para a API Whiser com a finalidade exclusiva de transcrição. Não armazenamos ou mantemos o áudio submetido para transcrição, nem os resultados da transcrição após serem entregues a você.

Security and Storage

É importante destacar que não armazenamos nenhum áudio enviado para transcrição em nossos servidores ou em qualquer outro lugar. Todo o processamento é realizado de forma temporária e transitória para fornecer a transcrição desejada, e os resultados são enviados diretamente de volta para você no Whatsapp.

Consent

Ao utilizar a extensão Audio to Text for Whatsapp Web, você expressamente consente com a transmissão do áudio para a API Whiser da OpenAI para fins de transcrição, conforme descrito nesta política de privacidade.

Changes to the Privacy Policy

Reservamos o direito de modificar esta política de privacidade a qualquer momento, mediante notificação aos usuários através de atualizações na extensão ou em nosso website.
